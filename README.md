# SQL Customer Sales Analysis

This project demonstrates the use of SQL for analyzing customer sales data and generating business insights. The analysis focuses on customer spending behavior, regional performance, product category profitability, and sales trends.

## Tools Used
- MySQL
- SQL

## Key Analyses
- Top customers by revenue
- Most profitable product categories
- Regional sales performance
- Customers spending above average
- Monthly sales trends

## Skills Demonstrated
- Data Analysis
- SQL Queries
- Aggregations
- GROUP BY
- Subqueries
- Business Intelligence
